Shortcode for embedding Liane forms

== Description ==

This plugin is for exclusive use with [Liane](https://liane.voto), an open source software for political innovation.

Use a simple shortcode to embed your campaign form into your WordPress site.

= Usage =

* On your Dashboard, go to Settings > Liane
* Fill the form with proper information provided in your Liane dashboard
* Use `[liane_form]` shortcode in any post or page to embed your form!
